# Demo

`plimsoll.tape` records a short terminal demo of Plimsoll gating two agent traces against the same policy and baseline: a clean trace passes with exit code 0, then a regressed trace fails with exit code 1 and a breakdown of findings by severity. The committed `demo.gif` is rendered from the `.tape` script with [charmbracelet/vhs](https://github.com/charmbracelet/vhs); regenerate it with `vhs demo/plimsoll.tape`. The recording is reproducible from the checked-in example fixtures.

`mcp-governor.tape` records the scripted MCP governor session (`scripts/build_mcp_governor_session.py`) driving the real `plimsoll-governor` stdio server through three pre-execution verdicts: an allow, a `tool_order` deny, and a `max_tokens` budget block. The committed `mcp-governor.gif` is rendered the same way — `vhs demo/mcp-governor.tape` — from an environment with the optional MCP extra installed (`pip install "plimsoll[mcp]"`). The session itself is deterministic; the committed transcript lives at `examples/mcp-governor-session/transcript.jsonl` (see `docs/MCP_DEMO.md`).
